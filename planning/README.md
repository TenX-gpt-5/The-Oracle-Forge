# AI-DLC Planning

This directory holds the lightweight AI-DLC workflow for Oracle Forge.

Use three artifacts per sprint:

1. An Inception document in [inception](/Users/gersumasfaw/week8_9/planning/inception)
2. A gate approval entry in [mob_session_log.md](/Users/gersumasfaw/week8_9/planning/mob_session_log.md)
3. An Operations document in [operations](/Users/gersumasfaw/week8_9/planning/operations)

Suggested sprint flow:

1. Copy [planning/inception/TEMPLATE.md](/Users/gersumasfaw/week8_9/planning/inception/TEMPLATE.md) into a dated sprint file.
2. Review and approve it during a mob session before coding.
3. Build only against definition-of-done items with evidence.
4. Record the approval gate in [planning/mob_session_log.md](/Users/gersumasfaw/week8_9/planning/mob_session_log.md).
5. Close the sprint with an operations note based on [planning/operations/TEMPLATE.md](/Users/gersumasfaw/week8_9/planning/operations/TEMPLATE.md).

Keep the process lightweight. The goal is to preserve decision quality and evidence, not to create bureaucracy.
